# Retail Sales Performance Dashboard (Excel)

## Project Overview

This project analyzes retail sales data using Microsoft Excel. The objective is to help management understand sales performance, profitability, customer behavior, and regional trends through an interactive dashboard.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Customer Segment Analysis
- Regional Sales Analysis
- Monthly Sales Trend
- Discount vs Profit Analysis
- Category-wise Profitability
- Interactive Slicers

---

## Skills Demonstrated

- Data Cleaning
- Data Transformation
- Business Analysis
- Dashboard Design
- KPI Reporting
- Data Visualization

---

## Business Insights

- West region generated the highest sales.
- Technology products contributed the highest profit.
- Sales peaked during November.
- Higher discounts reduced profitability.
- Consumer customers generated the highest revenue.
